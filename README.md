# STProof Protocol - 通用事件链上确权与价值捕获协议

## 项目简介
STProof (Spatial-Temporal Proof) 是一套面向「**事件**」的通用链上确权与价值捕获协议。
不同于传统 RWA（实物资产映射）模式，STProof 直接对**客观发生的历史事件**进行存证，而非实物本身。
通过多源证据交叉验证、可信时间戳、去中心化存储，将真实事件转化为数字世界不可篡改的永久记录。

**核心愿景：**
不确权物，只确权事。
凡发生，皆可存；凡存证，皆可信。

## 核心特性
- **确权对象：唯事件**
  以客观事件为核心，实物仅为现场载体。无实物亦可证，彻底规避监管风险。
- **平权存证**
  事实记录无资金门槛。任何人只需支付 Gas 费即可铸造 Event NFT，无需额外费用。
- **价值机制：引用税**
  事件价值由被相信、被使用的次数决定。每一次链上引用都是对真实的投票，引用费为发起人带来持续收益。
- **全场景适配**
  覆盖有实物、无实物、混合三类事件，协议具备无限扩展性。
