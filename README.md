# STProof Protocol
## 事件确权协议：不确权物，只确权事

### 🎯 核心理念
**我们认证事件，而非资产。**
证明存在于实物之中，亦存在于虚无之中。
凡发生，皆可存证；凡存证，皆可信任。

实物归实物世界，事件归区块链世界。NFT 只锁事件，不锁物品。

---

### 📖 协议概述
STProof（时空证明）是首个面向**事件**的通用确权协议。它解决了数字时代中真实世界客观事实难以验证的长期痛点，为历史事件构建了一道防AI伪造、防篡改的信任底层。

#### 核心特性
-   **事件中心化认证**：针对客观历史事件进行确权，完全规避 RWA 监管风险。
-   **存证零门槛**：任何人只需支付 Gas 费即可铸造事件 NFT，无额外门槛。
-   **引用税机制**：事件的价值由被引用和信任的次数决定，而非投机炒作。
-   **完全去中心化与开源**：基于以太坊构建，所有代码与文档均遵循 MIT 开源协议。
