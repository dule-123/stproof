STProof (Spatial-Temporal Proof) 是一套创新的区块链协议，旨在为物理世界和数字世界中那些无法通过标准化API验证的独特历史事件，提供去中心化的、不可篡改的链上确权。

在AI深度伪造技术泛滥的时代，我们致力于为“事实”本身建立一个可验证的永久锚点。

🌟 核心理念

• 不确权物，只确权事：实物归实物世界，我们锚定的是“事件”本身，而非任何关联的资产，彻底规避RWA（现实世界资产）监管风险。

• 事实的记录不应有资金门槛：任何人都可仅支付Gas费来铸造一个事件凭证，让历史记录变得平权。

• 价值源自影响力：引入首创的“引用税”模型，让一个事件的价值与其在链上被引用、被参考的次数挂钩，而不是二级市场的炒作。

🧩 核心创新

1.  事件确权：从“资产确权”范式转向“事件确权”范式。实物仅是事件的一个可选的证明载体。
2.  多源交叉验证：要求至少两个独立来源的证据（如GPS、官方记录、影像）相互印证，并通过社区或专业验证者网络进行验证，构建强信度。
3.  可扩展的事件模板：通过“事件模板”系统，支持社区为不同领域（艺术品创作、新闻存证、学术发现等）定义自定义的验证规则。
4.  合规且面向物理世界：明确不与实物资产所有权绑定，但支持通过可选“实物存留”机制连接物理世界，并为未来的物理验证（如碳14检测）预留接口。

🔧 技术实现

• 智能合约标准：ERC-721 (NFT) 扩展，支持 EIP-2981 版税。

• 核心功能：mint (铸造)， cite (引用)， verify (验证)。

• 证据存储：证据包经可信时间戳固化后，存储在IPFS/Arweave等去中心化存储网络上。

• 核心代码库：https://github.com/stproof-protocol/erc721-event (包含可部署的合约代码)。

🎯 应用场景

• 收藏品与历史记录：首日封实寄、艺术品创作过程、文物考古现场。

• 内容与知识产权：线上内容首次发布、学术优先权存证、新闻现场记录。

• 法律与商业：重要合同签署、现场活动公证、供应链关键节点记录。
