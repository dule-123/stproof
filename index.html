协议核心理念

以平权为核心，重新定义链上确权，让事实成为数字世界的可信基础设施

确权对象：唯事件

统一以「客观发生的历史事件」为确权核心，非实体资产或数字内容。实物仅为事件现场载体。

核心理念：平权存证

事实记录无资金门槛，任何人只需支付Gas费即可铸造Event NFT，无需额外费用。

价值机制：引用税

事件价值不依赖炒作，而是由被相信、被使用的次数决定。每一次引用都是对真实的投票。

生态定位：重型事实层

专注高价值重型事实锚定，需多源证据、物理验证、终极仲裁的事件皆可适配。

全场景事件类型

覆盖有实物、无实物、混合三类事件

有实物事件

示例：实寄封邮寄、艺术品创作、文物出土、瓷器烧制

实物仅作为事件发生的现场载体，协议对「事件」确权，而非实物本身。

无实物事件

示例：线上发布、链上投票、观点声明、学术发表

纯数字/行为类事件，无伴随实物，协议直接对事件进行多源存证与确权。

混合事件

示例：新闻现场、演出活动、法律存证、线下会议

融合物理现场与数字记录，实现全维度事实存证。

Not Proof of Asset, But Proof of Event
What Happens, Exists; What's Proven, Trusted

STProof (Spatial-Temporal Proof) is a universal on-chain event proof and value capture protocol.
Breaking traditional asset-mapping logic, it turns objective historical events into immutable permanent records through multi-source evidence, trusted timestamps, and decentralized storage.

Whitepaper

Core Concepts

Equity-centered, redefining on-chain proof, making facts trusted infrastructure

Proof Object: Event Only

Focus on objective historical events, not physical assets. Physical items are only event carriers.

Core Philosophy: Equitable Proof

Zero financial barrier for fact recording. Anyone can mint Event NFT with gas only.

Value Mechanism: Citation Royalty

Event value comes from trust and usage, not hype. Every citation is a vote for truth.

Ecosystem Position: Heavy Fact Layer

Focus on high-value heavy facts requiring multi-source evidence and physical verification.

Event Types

Covers tangible, intangible and hybrid events

Tangible Events

First Day Cover, Art Creation, Relic Excavation, Production

Physical items are only carriers. The protocol proves the event, not the item itself.

Intangible Events

Online Release, On-Chain Voting, Statement, Publication

Pure digital or behavioral events with no physical form, directly proven on-chain.

Hybrid Events

Live News, Performance, Legal Proof, Offline Conference

Combining physical scenes and digital records for full-dimensional fact proof.
