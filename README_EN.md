# STProof Protocol
A universal on-chain attestation and value-capture protocol for "Events". It transforms objectively occurred historical events into immutable, permanent records in the digital world through multi-source evidence cross-verification, trusted timestamps, decentralized storage, and on-chain anchoring.

---

## 🎯 Core Philosophy
**We certify events, not assets.**
Proof exists with physical objects, and even without.
All that happens shall be stored; all stored proofs shall be trusted.

Physical objects belong to the physical world. Events belong to the blockchain world. NFTs lock only events, not objects.

---

## 📖 Protocol Overview
STProof (Spatial-Temporal Proof) is the first universal protocol dedicated to **event certification**. It solves the long-standing problem of verifying non-standard, real-world facts in the digital age, building a trust layer for historical events that is resistant to AI forgery and tampering.

### Key Features
- **Event-Centric Certification**: Certifies objective historical events, not physical assets or digital content, completely eliminating RWA regulatory risks.
- **Zero Barrier for Fact Recording**: Anyone can mint an Event NFT by paying only gas fees, with no additional costs.
- **Citation Tax Mechanism**: The value of an event is determined by how many times it is cited and trusted, not by speculative hype.
- **Fully Decentralized & Open Source**: Built on Ethereum, all code and documents are open source under the MIT License.
