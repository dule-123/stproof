# STProof Protocol - Universal On-Chain Event Proof & Value Capture Protocol

## Project Introduction
STProof (Spatial-Temporal Proof) is a universal on-chain event proof and value capture protocol.
Unlike traditional RWA (Real-World Asset) models, STProof directly certifies **objectively occurring historical events**, not the physical objects themselves.
Through multi-source evidence cross-verification, trusted timestamps, and decentralized storage, real events are transformed into immutable permanent records in the digital world.

**Core Vision:**
Not proof of asset, but proof of event.
What happens, exists; what's proven, trusted.

## Core Features
- **Proof Object: Event Only**
  Centered on objective events, with physical items serving only as on-site carriers. Proof exists without physical items, completely mitigating regulatory risks.
- **Equitable Proof**
  Zero financial barrier for fact recording. Anyone can mint Event NFTs by paying only Gas fees, with no additional costs.
- **Value Mechanism: Citation Royalty**
  Event value is determined by the number of times it is trusted and used. Every on-chain citation is a vote for truth, generating continuous revenue for the initiator.
- **Full-Scenario Adaptability**
  Covering tangible, intangible, and hybrid events, the protocol offers unlimited scalability.
